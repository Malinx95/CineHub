Cinehub est le site crée dans le cadre de l'unité d'enseignement de développement web, 2020-2021 à CY cergy-paris université, par Maxime GRODET et Antoine QIU.
Ce site permet de rechercher des films et séries, afin d'afficher plus d'information.
Il permet de montrer les compétance que nous avons aquis en HTML, CSS et PHP.
Ce site est accesible via les urls suivant : http://grodet-maxime.alwaysdata.net/projet/, http://klbgr.fun/projet/ et http://cinehub.fr/
Vous pouvez le code source de notre site depuis les GitHub à l'url suivante : https://github.com/Malinx95/CineHub
